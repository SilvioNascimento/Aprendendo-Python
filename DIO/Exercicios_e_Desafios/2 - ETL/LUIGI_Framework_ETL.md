# FRAMEWORK LUIGI

## Documentação do LUIGI

Para acessar a documentação do framework Luigi (O qual não foi desenvolvido para ETL, mas funciona para este tipo de tarefa), acesse por este link:

- [Documentação do framework luigi](luigi.readthedocs.io/en/stable)

**Aviso:** Caso não tenha instalado o framework, execute o comando abaixo para instalar luigi na sua máquina local (execute no seu terminal)

- `pip install luigi`

---

## Execução do Framework

Após instalar o luigi, vá no terminal que você executou o comando de instalação e digite `luigid`.

Se fez isso parabéns, você fez com que o framework luigi ser acessado por uma porta na rede. É só copiar a URL abaixo, colar na barra de navegação do seu navegador e clicar *Enter*.

- **URL:** `http://localhost:8082/`
